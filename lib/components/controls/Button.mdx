# Button
*A pretty button*

## Props
*Extends `Element`*

|Name|Type|Default|Description|
|----|----|-------|-----------|
|isRaised|`boolean`|`true`|Shows a translate-to-top animation|

## Usage
import {Playground} from "docz";

import Button from "./Button";

<Playground>
  <div>
    <h2>Colors</h2>
    <Button bg="primary" c="light" mr={12}>Lorem ipsum</Button>
    <Button bg="secondary" c="light" mr={12}>Lorem ipsum</Button>
    <Button mr={12}>Lorem ipsum</Button>
    <Button mb={32} c="primary">Lorem ipsum</Button>

    <h2>Unraised</h2>
    <Button mb={32} isRaised={false} bg="primary" c="light" mr={12}>Lorem ipsum</Button>

    <h2>Disabled</h2>
    <Button isDisabled bg="primary" c="light" mr={12}>Lorem ipsum</Button>
  </div>
</Playground>
